🚀 Ion Beam Shepherd (IBS) for Space Debris Mitigation
Overview

The Ion Beam Shepherd (IBS) project is a MATLAB/Simulink-based simulation of an advanced space debris removal system. The concept uses a dedicated spacecraft equipped with ion thrusters to generate a highly collimated ion beam that transfers momentum to space debris without physical contact. This contactless approach enables controlled orbital modification and safe deorbiting of inactive satellites and debris objects in Low Earth Orbit (LEO).

Objectives
Simulate orbital dynamics of a debris object and an IBS spacecraft.
Model ion-beam-induced momentum transfer.
Analyze debris orbital decay and deorbiting behavior.
Develop guidance and control algorithms for debris tracking.
Visualize spacecraft–debris interaction in a 3D orbital environment.
Investigate power-efficient deorbiting strategies for future missions.

Future Scope

Future work focuses on implementing an intelligent thrust scheduling mechanism where the ion beam is activated only near the orbital apogee (farthest point from Earth). Since debris velocity is minimum at apogee, orbital corrections can be achieved more efficiently, reducing power consumption and extending mission lifetime.

🚀 Ion Beam Shepherd (IBS) for Space Debris Mitigation
📖 Overview

The Ion Beam Shepherd (IBS) project is a MATLAB/Simulink-based simulation of a contactless space debris removal system. The concept uses a CubeSat equipped with ion thrusters to generate a directed ion beam that transfers momentum to debris objects, gradually modifying their orbit and enabling safe deorbiting without physical contact.

This project focuses on modeling orbital dynamics, spacecraft-debris interaction, ion beam momentum transfer, guidance and control systems, and deorbiting analysis in Low Earth Orbit (LEO).

🎯 Objectives
Simulate the orbital motion of a CubeSat and space debris.
Model ion beam force and momentum transfer.
Implement guidance and control algorithms for debris tracking.
Analyze orbital decay and deorbit trajectories.
Visualize spacecraft and debris motion in a 3D environment.
Investigate power-efficient debris removal strategies.
