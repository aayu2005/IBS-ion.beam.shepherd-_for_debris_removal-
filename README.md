# IBS-ion.beam.shepherd-_for_debris_removal-
🚀 **Ion Beam Shepherd (IBS) for Space Debris Mitigation** — A MATLAB/Simulink-based simulation of a CubeSat using ion-beam propulsion for contactless space debris removal. Features orbital dynamics, debris tracking, PD control, momentum transfer modeling, deorbit analysis, and energy-efficient thrust strategies for sustainable space operations. 🌍
